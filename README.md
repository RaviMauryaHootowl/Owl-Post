# Owl Post

Owl Post - API Testing Tool (in Development)<br/>
A Postman Clone

## Installation and Run

At your Terminal Run-

Step 1:
```
 npm install 
``` 

Step 2:
```
 npm start 
``` 

Step 3:
Go to your browser to this link - [localhost](localhost:3000) to see the app.<br>
